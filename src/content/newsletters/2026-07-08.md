### GitHub's New AI Agent Has a Security Problem. Researchers Just Proved It.

Noma Security discovered a critical flaw in GitHub's Agentic Workflows that lets anyone trick the AI agent into leaking private repository data. By posting a crafted issue in a public repo, an attacker can make the agent fetch and expose files from private repos within the same organization. GitHub's guardrails were supposed to prevent this, but adding the word 'Additionally' bypassed them entirely.
