### Claude Code Burns Through 33,000 Tokens Before You Even Type a Word

A Systima study found that Anthropic's Claude Code sends roughly 33,000 tokens of system prompts, tool schemas, and injected scaffolding before processing a single user request, compared to OpenCode's 7,000 tokens. The overhead includes 27 tool definitions, orchestration suites, and reminder blocks that grow mid-session, with cache re-writes costing up to 54 times more than competitors. Real-world setups with instruction files and MCP servers can hit 75,000 to 85,000 tokens before the first keystroke.

[Source: Systima →](https://systima.ai/blog/claude-code-vs-opencode-token-overhead)
